# AMNKART (Electronics Storefront)

Static, dependency-free ecommerce storefront (HTML/CSS/JS) with:
- Home, collections, product detail, search, cart, and mock checkout
- Cart stored in `localStorage`
- INR pricing (₹) and simple filters/sort on collections

## Run

Open `index.html` in a browser.

If your browser blocks local JS on `file://`, use any simple static server (optional), for example:
- VS Code “Live Server”
- Any local static server you already have installed

## Edit products

Update catalog data in:
- `assets/data.js`

## Pages

- `index.html` (Home)
- `collections.html?handle=chargers`
- `product.html?handle=bolt-65w-gan-charger`
- `search.html?q=charger`
- `cart.html`
- `checkout.html`
